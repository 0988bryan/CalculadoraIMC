# Calculadora de IMC (Índice de Masa Corporal)

## Descripción
Esta aplicación es un proyecto académico desarrollado para la carrera de IT. Permite al usuario calcular su Índice de Masa Corporal ingresando su peso y altura de manera sencilla y rápida.

## Características principales
* Cálculo preciso del IMC basado en la fórmula estándar.
* Interfaz intuitiva desarrollada con **Jetpack Compose**.
* Navegación fluida entre pantallas usando **NavHost**.
* Diseño responsivo y personalizado con **Material Theme 3**.

## Tecnologías utilizadas
* **Lenguaje:** Kotlin
* **Framework:** Jetpack Compose
* **Arquitectura:** MVVM (Modularizada)
* **Control de versiones:** Git y GitHub
